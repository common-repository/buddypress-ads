== Description ==

The developement of this plugin is paused ! This plugin does not support buddypress 1.2


This plugin will allow you to publish ads through your buddypress site(with wordpress mu) Supports multiple ad networks including adsense , adbrite and many more. Just replace the ad code in the file bp-custom.php with your own ad code.

Supported Ad Networks :

* Google Adsense

* Adbrite

* Bidvertiser

* Kontera

* All other





== Installation ==


The developement of this plugin is paused ! This plugin does not support buddypress 1.2


The installation is pretty simple as follows :



--> Replace the ad code in bp-custom.php file with your ad code.

---> Upload the bp-custom.php into your /wp-content/plugins directory

---> Done !
